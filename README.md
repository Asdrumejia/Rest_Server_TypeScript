# Nota
Rest_Server_TypeScript
 

# Comandos

```
Ejecutar npm install ó npm i para instalar o reconstruir los módulos de Node

Ejecutar tsc para compilar, transpilar o traducir los archivos a JavaScript

Ejecutar nodemon dist/app ó nodemon dist/app.js para correr la aplicacion 

Ejecutar tsc --watch (en terminal alterna) para que TypeScript detecte cualquier cambio en nuestra app de Node 

```
